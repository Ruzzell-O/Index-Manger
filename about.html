    About us

    These are the developers who operated and formed the Index Manager as you see now. With the goal to ensure a reliable interface through semantic code that produces the necessary functions for the program to be operational, and provide satisfaction towards the user's experience. They were the ones responsible in forming the tool that will convenience HRM staff in their recording needs.

    Mission

    The team's mission is to create a web application with functionalities that are applied using the team's own experiences and knowledge and create something that works other than the user interface. The team will also make the application flexible to ensure a stable working environment with the data in just one place and have an efficient operation. In addition, the team also aims to create the application as an alternative in storing records of applicants, bringing efficiency to the users.
