    how it works
    This section contains information about the different functions
    and capabilities of the website. It is recommended for the user
    to acknowledge this page first before using the program. It is
    in the developers' best interest to provide a seamless, reliable,
    and efficient database for HRM needs.

    Functionalities

    New Applicant


    This button or function transports the user to the form section to encode the details for the records of applicants.

    Form Section


    This is where the user encodes the details for their applicants through varying requirements depending on their company's needs. In this case, the developers provided a static list of details needed to be encoded by the user as per the requirements of providing the standard information to be listed for the database of records.

    Search


    The search bar allows the user to find the list of names they're looking for if the user's table has too many applicants.

    Pagination


    Pagination occurs when the database has too many applicants to be shown within the website. The solution provided is that the extra applicants will be distributed to another page, and the program provides the necessary pages for the user to navigate their records.

    Light/Dark Mode


    This function allows the layout of the website to change from light theme to dark theme for the comfort of the user.

    Update Applicant


    This function provides a pop-up window that enables the user to update existing information within the records of the table. This specifies which row the user wants to update and the new value will change or modify existing records in a table.

    Profiling Applicant


    This function provides a pop-up window that showcases the full information regarding the applicant's records encoded within the database if not present within the table.

    Delete Applicant


    This function deletes the selected record from the database.

    View Documents


    This function enables the user to view the stored documents encoded within the applicant's records. The documents uploaded here should only be PDF files as to enable the browser the user's using to render, assess, and download the specified file in the program.

    Photo and Document Section within Forms


    This function is used to specify which column you want to update and the new value will change or modify existing records in a table.
